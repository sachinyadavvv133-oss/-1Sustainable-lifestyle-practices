# Sustainable Lifestyle Practices

A simple static website with practical tips and resources to live a more sustainable lifestyle.

How to use:
- The site is static and lives at the repository root.
- To publish, enable GitHub Pages for this repository and select the `main` branch (root).

Files added:
- `index.html` — the website HTML
- `styles.css` — site styles
- `script.js` — small JS for navigation and smooth scrolling

Contributions welcome: open issues or PRs to improve content, accessibility, translations, or add assets.
